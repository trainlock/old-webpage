Siege Siege Siege

Course: TNM085 - Modelling project
This course consisted of a simulation project.
Project members: 7

The goal of the bachelor project was to develop a virtual reality game with physical game cards as controllers. The aim was to shorten the distance between the player and the game by having real objects interact with the virtual world.
     Unity was used as a game engine to develop the game, Maya for animation and modelling, Vuforia for tracking the game cards and HTC Vives VR-glasses to visualise the game. During development the agile framework Scrum to structure the work and each sprint was two weeks long.
     In the game, the player’s goal was to eliminate the opponent's home base by placing the physical game cards on a digital battlefield and summon minions to attack the opponent. The player only have to keep track on three physical cards which are dynamically updated in the game to randomise what type of minion the player can summon. Three different type of minions exists in the game and the battle system follows the rock-scissors-paper mechanics.
     The result was the actual game where the primary focus was on the UI and UX for the game cards. Physical cards were limited to three since more cards can be hard for the player to hold or cause confusion. Only low-polygon models was used in the game to lower the performance requirements.
