Shape Matching Deformer

Course: TNCG13 - SFX Tricks of the Trade
This course consisted of a simulation project.
Project members: 2

The aim of this project was to create a Python plugin to Maya that deforms objects using shape matching. The deformation should turn the objects into a jelly-like substance. This project uses the method proposed by Müller et al. in their paper ”Meshless Deformations Based on Shape Matching” (2005) to simulate the deformation. Everything was written in Python that is then loaded into Maya.
     To deform the object several different user-controlled attributes were added, such as mass, stiffness and elastic forces. The deformation varies when the attributes are altered. Vertices of meshes are simulated individually as particles and after each time step, the original shape gets matched against the deformed one. Three different deformations were implemented, Rigid Body Deformation, Linear Deformation and Quadratic Deformation, and the one shown in the video id the Quadratic Deformation. For the deformation, the best quadratic transformation is found and then applied to the original shape of the vertices which then is translated towards their respective transformation position.
     The end result is a mesh that gets transformed into a jelly-like substance in real-time.
