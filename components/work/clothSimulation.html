Cloth Simulation

Course: TNM085 - Modelling project
This course consisted of a simulation project.
Project members: 4

The basics of the algorithm for simulating mass-spring systems was first implemented in MATLAB, to get the math and physics working. The goal was to simulate cloth that moved realistically. For better stability in the mass-spring system, diagonal springs were added to the system and Euler integration and RK4 was used for the calculations.
