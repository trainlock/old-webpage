Modelling and Animation

Course: TNM079 - Modelling and Animation
This course consisted of serveral graded labs, with written reports on most of them.

Mesh Data Structures

In this lab the Half-edge data structure to be able to find neighbouring faces was implemented. This method allows neighbouring access to vertices and faces. Methods for calculating normals, area, volume, curvature and genus are presented.
Full Report: Mesh Data Structures

Splines and Subdivision

This lab session covered splines and subdivision methods. The Loop subdivision scheme was implemented as well as an adaptive subdivision scheme based on mean curvature. The mesh subdivision is based on Loop subdivision which subdivides each triangle into four new ones based on weighted sums of the old vertex positions.
Full Report: Splines and Subdivision

Implicit Surfaces and Level Sets

This part of the lab session was about implicit surfaces and Level sets. The purpose of the lab was to get a better understanding of how to work with implicit functions and different representations, such as the general quadric function formulation.
    Implicit quadric surfaces such as ellipsoids, cones and planes where implemented. Three boolean operations, union, intersection and difference where also implemented.
    For the level set implementation, operations such as erosion and dilation was implemented using Godunov scheme by using an upwind scheme. Parabolic diffusion and hyperbolic advection was implemented and calculated with a central difference scheme for the diffusion and an upwind scheme for the advection.
Full Report: Implicit Surfaces and Level Sets

Eularian Fluid Simulation

Basic functionalities for a stable Eularian based fluid solver, based on the Navier-Stokes equation, was implemented. The basic functionalities where external forces, Dirichlet boundary condition and the projection step for volume preservation. For self-advection semi-Lagrangian interpolation was used.
Full Report: Eularian Fluid Simulation
